# Keyboard Shortcuts

Cronboard is fully keyboard-driven. Below is a complete reference of all available key bindings.

---

## Global

| Key | Action |
|---|---|
| `q` / `Ctrl+Q` | Quit Cronboard |
| `Tab` | Switch focus to the next panel |

---

## Cron Table (Local & Remote)

### Navigation

| Key | Action |
|---|---|
| `j` / `↓` | Move cursor down |
| `k` / `↑` | Move cursor up |
| `h` / `←` | Move cursor left |
| `l` / `→` | Move cursor right |

### Cron Job Actions

| Key | Action |
|---|---|
| `c` | Create a new cron job |
| `e` | Edit the selected cron job |
| `D` | Delete the selected cron job |
| `p` | Pause / resume the selected cron job |
| `r` | Refresh the cron job list |

### Search

| Key | Action |
|---|---|
| `/` | Open search prompt |
| `n` | Jump to next search match |
| `N` | Jump to previous search match |
| `Escape` | Clear search / close modal |

---

## Servers Panel

| Key | Action |
|---|---|
| `a` | Add a new remote server |
| `c` | Connect to the selected server |
| `d` | Disconnect from the current server |
| `D` | Delete the selected server |
| `j` / `k` | Navigate the server tree |

---

## Forms & Modals

| Key | Action |
|---|---|
| `Tab` | Accept path autocompletion suggestion |
| `Escape` | Close the current modal without saving |
